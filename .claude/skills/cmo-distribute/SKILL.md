# Skill: cmo-distribute

## Purpose

Plan and execute content distribution across platforms.

## Trigger

Use when content is ready to publish or when planning a distribution schedule.

## Workflow

1. Read `context/SOCIAL_STRATEGY.md` for platform strategy
2. Read `context/CONTENT_PIPELINE.md` for content ready to distribute
3. Adapt content for each target platform
4. Create platform-specific versions in `drafts/`
5. Once published, move finalized content to `published/`
6. Update `context/CONTENT_PIPELINE.md` to reflect new status

## Inputs

- Content to distribute (from `drafts/` or user-provided)
- Target platforms
- Timing preferences

## Outputs

- Platform-adapted content versions
- Distribution plan
- Updated pipeline status
