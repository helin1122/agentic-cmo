# Skill: cmo-cowork

## Purpose

Collaborative content creation sessions. Work with the user to ideate, draft, and refine content.

## Trigger

Use when the user wants to create or iterate on content together.

## Workflow

1. Read `context/VOICE.md` for tone and style guidelines
2. Read `context/CONTENT_PIPELINE.md` for current pipeline state
3. Collaborate with the user on content ideation or drafting
4. Save drafts to `drafts/` with descriptive filenames
5. Update `context/CONTENT_PIPELINE.md` with new items

## Inputs

- Topic or content brief from the user
- Target platform (Twitter/X, LinkedIn, blog, etc.)
- Any reference material or inspiration

## Outputs

- Draft content saved to `drafts/`
- Updated content pipeline
